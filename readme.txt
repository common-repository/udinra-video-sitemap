== Installation ==

The plugin can be installed like any other plugin.You can search from WordPress Dashboard (Udinra Video Sitemap) and click install to install it.You can also download it and upload it to your server wp-content/plugins directory and extract the zip file.

== Frequently Asked Questions ==

You need to submit your-domain-name.com/sitemap-index-video.xml to your Google Webmasters (Google Search Console) and other webmasters site.If plugin faces any problem in creating sitemap file create sitemap-index-video.xml file in root directory and set access as 664.
